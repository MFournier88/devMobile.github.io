React Native & Expo

Installation

Allez sur votre container sur le serveur


sudo apt install curl

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash
source ~/.bashrc
nvm install 22
# Vérifiez l'installation
npm -v
node -v

npm install -g expo-cli

npx create-expo-app@latest -t

# Choisissez blank comme template

cd 

npx expo start

Si tout c'est bien passée vous devriez voir un code QR. Il est maintenant temps d'installer Expo Go sur votre appareil mobile.

Après l'installation, vous pouvez scanner le code QR et voir l'application de base se déployer sur votre mobile.
